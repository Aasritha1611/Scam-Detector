# 🔍 AI Job & Internship Scam Detector

An AI-powered web application that helps users identify fake job and internship offers by analyzing text using Machine Learning and Natural Language Processing (NLP).

Users can paste any job message into the website, and the system will classify it as **Scam** or **Genuine**, along with a confidence score.

---

## 🚀 Features

- 🧠 Machine Learning-based scam detection  
- 📝 NLP-based text analysis  
- ⚡ Real-time prediction using FastAPI  
- 🌐 User-friendly web interface  
- 📊 Scam probability and trust score  
- 🔗 Deployed and accessible online  

---

## 🧠 How It Works

1. User enters a job/internship message  
2. Text is processed using feature extraction  
3. Machine learning model analyzes patterns  
4. API returns prediction  
5. Result is shown as Scam or Genuine with score
